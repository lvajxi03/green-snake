A Green Snake

Download 7-zip archive

About

Game written as a tribute to original Oil's Well from the eighties, prototyped in Python programming language and PyQT library and finally made in C++ for Windows using pure WinAPI.

Installation

This program is distributed as 7-zip archive with single executable file inside, ready to run and play. You can extract the archive anywhere you want, no extra directory structure is required.

The default font is awful and it's better to download and install C64 Pro (for instance, here)

User manual

Menu navigation includes cursor keys for selecting options and Enter to confirm your choice or left mouse button on selected option.

Switching languages is possible only with left mouse button.

This program does not create nor read any files, its own settings (last used game mode and hiscores table) are stored in system registry (individually per user)

Game

Your goal is to complete all game levels (currently there are six of them) by controlling the green snake and collecting all available objects like yellow squares, purple diamonds and gold coins. For every object collected you get some points.

Level is completed when all objects are collected. You can see at the bottom status line that you have to hurry, otherwise you will lose. Every level has its own time counter (it grows with level number and game mode)

You need to use cursor keys to move snake. Snake is not able to eat itself. Moreover, you can use backspace key to move back.

You can eat creatures that moves from left to right and back, however they shall not collide with snake's body.

A Ghost that appears sometimes, can hit snake's body, but you should avoid eating it (a collision will be also detected)

Three collisions means you lost.

When you collect blue pill the level counter is frozen for 20 seconds, also all creatures stop. Collecting another blue pill increases the freeze counter back to 20 seconds.

No Time Limit Mode

(No Time Limit Mode is proposed by my son, Antoni, who is 8 now)

When you collect a gold coin, new sword appears in a random place. You can collect them safely and the status line will show how many of them you have. This mode has collisions switched off and no time limit (yes, seriously)

This mode probably will be removed from options and for sure will not be developed at all.

Misc

Source code of this program was written in Emacs editor, graphics were made in Inkscape and GIMP. Everything was compiled, linked, debugged (many times!) and profiled with MSYS2 collection of utilities.

Kudos to my son, Antoni, for all the feedback, dozens of ideas and many hours of testing.

Prototype was made on Raspberry Pi computer with Ubuntu Linux installed, also with Emacs editor, GIMP and Inkscape as well, however Python 3 was main programming language and PyQt 5 delivered all the windowing and user interface.
